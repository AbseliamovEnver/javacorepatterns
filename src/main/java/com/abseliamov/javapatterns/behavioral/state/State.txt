State (Состояние) - объект может изменять свое поведение в зависимости от его состояния.

Допустим, в графическом редакторе вы выбрали инструмент «Кисть». Она меняет своё
поведение в зависимости от настройки цвета: т. е. рисует линию выбранного цвета.

Очень похож на паттерн Strategy

Примеры применения паттерна проектирования State в JDK:
    javax.faces.lifecycle.LifeCycle#execute()
    (контролируется FacesServlet, поведение меняется в зависимости от фазы (состояния) JSF)

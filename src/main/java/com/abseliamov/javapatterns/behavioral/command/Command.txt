Command (Команда) - интерфейс команды объявляет метод для выполнения определенного действия.

Шаблон «Команда» позволяет инкапсулировать действия в объекты. Ключевая идея — предоставить
средства отделения клиента от получателя.

Вы пришли в ресторан. Вы (Client) просите официанта (Invoker) принести блюда (Command).
Официант перенаправляет запрос шеф-повару (Receiver), который знает, что и как готовить.

Другой пример: вы (Client) включаете (Command) телевизор (Receiver)
с помощью пульта (Invoker).

Примеры применения паттерна проектирования Command в JDK:
    Все реализации  java.lang.Runnable
    Все реализации  javax.swing.Action

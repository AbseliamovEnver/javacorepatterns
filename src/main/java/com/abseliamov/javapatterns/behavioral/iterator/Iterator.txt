Iterator (Итератор) - последовательно осуществляет доступ к элементам объекта коллекции,
не зная его основного представления.

Хороший пример — радиоприёмник. Вы начинаете с какой-то радиостанции, а затем перемещаетесь
по станциям вперёд/назад. То есть устройство предоставляет интерфейс для итерирования
по каналам.
